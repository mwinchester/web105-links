Images

It is easy to use an image in place of text for a link. You just nest the image tag inside of the anchor tag where you would normally put text.

Clicking the star will take you to the course page for Web 115. It used to be common to create image buttons for links, now it is standard practice to use CSS to create button effects.

<a href= "http://web115.mccdgm.net/">
<img src="star.png" alt="star" style="float: left; padding:15px;">
< /a >
